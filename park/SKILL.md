# Park

Output a large plaintext box-drawing banner capturing the
current state of work.

Read memory files and task list for context. No questions.

The banner must be self-contained (no "as we discussed"),
show commands not descriptions, and state decisions as facts.
Plaintext only — readable in any monospace context.

`/park [topic]` focuses on a specific thread.
